# Evaluation

Use this folder to define how tool choices, workflows, and agent recommendations will be evaluated.

## Evaluation Targets

- Does the workflow order make sense?
- Are required inputs available?
- Are required metadata available?
- Are tool assumptions satisfied?
- Are outputs compatible with downstream tools?
- Does the recommendation identify uncertainty?
