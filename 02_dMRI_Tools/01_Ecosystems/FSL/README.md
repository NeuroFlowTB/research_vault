# FSL

## Role

FSL is a classic neuroimaging ecosystem with widely used diffusion MRI preprocessing and analysis tools.

## Key Tools

| Tool | Purpose |
|---|---|
| TOPUP | Susceptibility distortion correction |
| EDDY | Eddy current and motion correction |
| DTIFIT | Diffusion tensor fitting |
| TBSS | Tract-based spatial statistics |
| BedpostX | Bayesian modeling of diffusion parameters |
| ProbtrackX | Probabilistic tractography |
