# MRtrix3

## Role

MRtrix3 is a diffusion MRI ecosystem focused on advanced modeling, tractography, and flexible command-line workflows.

## Key Areas

- preprocessing support
- response function estimation
- fiber orientation distribution modeling
- tractography
- connectomics
