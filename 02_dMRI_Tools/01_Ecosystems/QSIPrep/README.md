# QSIPrep

## Role

QSIPrep is a BIDS-based workflow orchestration ecosystem for diffusion MRI preprocessing.

It can coordinate tools from multiple ecosystems, including FSL, MRtrix3, ANTs, and DIPY.

## Key Ideas

- reproducible preprocessing
- BIDS input structure
- automated workflow construction
- structured outputs
- interoperability across tools
