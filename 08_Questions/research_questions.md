# Research Questions

## Neuroimaging

- How is diffusion MRI different from standard structural MRI?
- Which artifacts are most important in dMRI preprocessing?
- When is TOPUP required?
- What inputs does FSL EDDY require?
- How do eddy current distortions differ from susceptibility distortions?
- Which preprocessing steps are handled by QSIPrep automatically?
- How do FSL, MRtrix3, TORTOISE, and QSIPrep overlap?

## Agent Reasoning

- What does it mean for an AI agent to reason over a preprocessing workflow?
- How can tool capabilities be represented in a structured knowledge base?
- How should the agent detect missing metadata?
- How should the agent explain uncertainty?
- What counts as interoperability between neuroimaging ecosystems?
- How can retrieval be evaluated in this thesis?

## Evaluation

- What benchmark tasks can test whether the agent understands preprocessing order?
- Can the agent identify invalid workflow sequences?
- Can the agent recommend tools based on available inputs?
- Can the agent distinguish known facts from missing information?
