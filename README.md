# Research Vault

This vault is the cognition framework for the bachelor thesis.

## Thesis Focus

The project studies diffusion MRI preprocessing and how knowledge-grounded AI agents can reason about neuroimaging workflows.

Core themes:
- diffusion MRI preprocessing
- preprocessing pipeline reasoning
- interoperability between neuroimaging ecosystems
- structured knowledge bases
- self-knowledge-aware retrieval
- workflow dependencies and ordering constraints

## Main Ecosystems

- FSL
- MRtrix3
- TORTOISE
- NiPreps / QSIPrep

## Current Learning Path

1. Understand MRI fundamentals.
2. Build intuition for diffusion MRI.
3. Understand why preprocessing is necessary.
4. Map preprocessing problems to correction steps.
5. Compare neuroimaging tool ecosystems.
6. Design an AI agent that can reason over workflow knowledge.
